1. 전위 증가 연산자와 후위 증가 연산자의 차이는?
-> 전위 증가 연산자는 실행문에서 값이 먼저 증가/감소가 일어나고
    후위 증가 연산자는 실행문에서 값이 마지막에 증가/감소가 일어난다.

2. 아래의 비트 연산자 4가지를 설명하시오.
 &  ->  비트 단위로 AND 연산을 한다.  ex) n1 & n2;
  |  ->  비트 단위로 OR 연산을 한다.  ex)  n1 | n2;
 ^  ->  비트 단위로 XOR 연산을 한다.   ex) n1^n2;
 ~  ->  피연산자의 모든 비트를 반전시켜서 얻은 결과를 반환한다.  ex) ~n;


3. 쉬프트 연산자에 대하여 설명하시오.
  x << y : 정수 x의 각 비트를 y만큼 왼쪽으로 이동시킨다. (빈자리는 0으로 채워진다.)
  x >> y : 정수 x의 각 비트를 y만큼 오른쪽으로 이동시킨다. 
             (빈자리는 정수 a의 최상위 부호비트와 같은 값으로 채워진다.)
  x >>>y : 정수 x의 각 비트를 y만큼 오른쪽으로 이동시킨다. (빈자리는 0으로 채워진다.)


4. 조건문의 3가지 종류를 나열하고 설명하시오.
 if문 : 조건이 충족되면 실행되는 문장.
 else if문: if문 외에 또 다른 조건이 충족되면 실행되는 문장.
 else문 : if나 else if문의 조건이 충족되지 않으면 실행되는 문장.


5. 삼항연산자란?
 ->    조건문   ?   a   :    b ;          : 조건이 참일 경우 a 거짓일 경우 b 이다. 
         (조건)      (참)    (거짓)   

6. switch 문이란?
-> switch문은 어떤 변수의 값에 따라서 문장을 실행할 수 있도록 하는 제어문이다.
