1. 상수란 무엇이며 변수와의 차이는?
-> 상수란 변수의 반대로 변하지 않는 값이다.
    키워드 final 선언이 붙어있는 변수.
    ex) final int a = 100; 


2. 초기화란?
-> 객체를 선언하고 값을 '최초'로 할당하는 것.



3. 리터럴이란?
-> 변수에 넣는 변하지 않는 '데이터'를 의미한다.
    변수나 상수는 메모리에 할당된 공간이라면
    리터럴은 이 공간에 저장되는 값이라고 할 수 있다.
    (근데 실무에서 상수와 리터럴은 비슷한걸로 보지 딱히 나누지 않는다.)
    ex) 5와 7은 정수형 리터럴, 3.3과 4.5는 실수형 리터럴.     


4.아래의 내용이 에러가 나는 이유와 수정 방법은?

java
System.out.println(3147483647 + 3147483648); //에러남

-> 정수형 범위는 -21억부터 +21억까지이므로 범위를 벗어난다. 
    l 또는 L 을 붙여서 long형 상수로 표현해야한다. 



5. 자동 형변환과 명시적 형변환에 대하여 설명하시오.
-> 자동 형변환: 프로그램 실행 도중에 자동적으로 타입 변환이 일어나는 것. (작은 데이터타입 -> 큰 데이터타입)
-> 명시적 형변환: 프로그래머가 강제적(명시적)으로 타입을 변환하는 것. (큰 데이터타입 -> 작은 데이터타입)



6.아래의 에러가 나는 이유를 설명하고 수정 하시오.

int a = 3;
int b = 4;
	
double result = a / b;  // <- (수정) double result = (double)a / b;

-> int - int 산술연산은 int로 결과값이 나오기 때문에 변수 하나를 double형으로 명시적 형변환을 해주어야 한다.



7. 이스케이프 시퀀스의 종류를 나열하고 기능을 설명하시오.
-> 정의: 문자나 키보데에서 직접 입력할 수 없는 문자를 표현 하기 위한 방법.

           - \r: 복귀
           - \': 작은 따옴표, \": 큰 따옴표
           - \\: \문자 



8. 아래의 출력 결과를 확인하고, 130이 나오는 이유를 설명


System.out.println('A'); 
System.out.println('A' + 'A'); 

-> char 더하기 char 는 결과값이 int형으로 변환되어서 나온다. 
    아스키코드표에 따라서 65 + 65가 되어서 130이 나온다.

13.아래와 같이 출력값이 나오는 이유를 설명하시오.

public class Hello {
	public static void main(String[] args) {
		int ch = '1';
		int num = 1;		
		System.out.println(ch);
		System.out.println(num);   	    
	}
}


/* 
출력결과
49
1
*/

-> int형 ch에 문자 '1'이 대입되어서 아스키코드표에 따라서 '1'은 십진수로 49이므로 ch에 49가 저장되기 때문이다.



14. ASCII(아스키 코드)란?

    *문자 인코딩은 문자들의 집합을 부호화하는 방법이다.
